# USAID SME-Related Assistance Analysis

## Project Overview

This repository contains materials for the BIDA 670 Business Analytics project analyzing whether USAID SME-related assistance is associated with measurable country-level economic outcomes over time.

The project originally focused on identifying the most effective individual SME loans, but the scope was refined because borrower-level outcome data was not consistently available. The final project uses a country-year analysis framework to compare USAID SME-related obligations and disbursements with GDP and GDP growth indicators.

## Business Question

Is USAID SME-related assistance associated with improved country-level economic outcomes over time?

## Data Sources

The project used publicly available data from sources such as:

- ForeignAssistance.gov
- USAspending.gov
- IATI Registry
- World Bank Open Data

## Methods Used

The analysis included:

- Data cleaning and preparation
- Country-year aggregation
- Descriptive statistics
- Correlation analysis
- Ordinary least squares regression
- Business interpretation and decision-support reporting

## Key Findings

The analysis found weak same-year statistical relationships between USAID SME-related funding and GDP growth. This does not mean the assistance lacks value. Instead, it suggests that country-level GDP growth is too broad to serve as the only measure of SME assistance impact.

The project is best understood as a monitoring and decision-support framework rather than a causal impact study.
