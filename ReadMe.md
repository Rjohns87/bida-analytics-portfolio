**Featured Project**



**BIDA 670:** USAID SME-Related Assistance Analysis

This project evaluated whether USAID SME-related assistance was associated with country-level economic outcomes over time. The project used public foreign assistance data and World Bank economic indicators to build a country-year analysis framework.



**Tools Used:** Excel, Python, regression analysis, correlation analysis, data cleaning, business report writing  

My Contribution: Literature review, data quality review, analysis interpretation, report development, and project framing  

Key Finding: The analysis found weak same-year statistical relationships between USAID SME-related funding and GDP growth, suggesting that country-level GDP is too broad to serve as the sole measure of SME assistance impact.
